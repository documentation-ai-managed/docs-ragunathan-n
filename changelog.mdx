## Upgrade Instructions

Follow these steps to upgrade Ragunathan N to the latest version.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command in your project root.
    ```bash
    npx ragunathan-n --version
    ```
  </Step>
  <Step title="Update Package" icon="download">
    <Tabs>
      <Tab title="npm" icon="npm">
        ```bash
        npm install ragunathan-n@latest
        ```
      </Tab>
      <Tab title="yarn" icon="yarn">
        ```bash
        yarn add ragunathan-n@latest
        ```
      </Tab>
      <Tab title="pnpm" icon="package">
        ```bash
        pnpm add ragunathan-n@latest
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Verify Installation" icon="check-circle">
    Test a sample MDX page to ensure components render correctly.
  </Step>
</Steps>
