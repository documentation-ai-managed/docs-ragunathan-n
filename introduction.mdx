## Overview

Ragunathan N provides a centralized space to organize, manage, and publish your project documentation. You create custom documentation sites with MDX support, version control integration, and powerful search. Build professional docs for APIs, SDKs, or user guides without managing complex hosting.

This platform handles rendering, SEO, and analytics, so you focus on content. Start with a template, add pages, and deploy instantly.

## Get Started

Follow these steps to create your first documentation space:

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at [app.ragunathann.com](https://app.ragunathann.com) and verify your email.
  </Step>
  <Step title="New Project" icon="plus">
    Click "New Space" and select a template for API docs or general projects.
  </Step>
  <Step title="Install CLI" icon="download">
    Use the CLI for local development:
    
    <CodeGroup tabs="npm,yarn">
      ```bash
      npm create ragunathan-app@latest my-docs
      cd my-docs
      npm run dev
      ```
      ```bash
      yarn create ragunathan-app my-docs
      cd my-docs
      yarn dev
      ```
    </CodeGroup>
  </Step>
  <Step title="Publish" icon="rocket">
    Commit changes and push to deploy. Your site goes live at `yourspace.ragunathann.com`.
  </Step>
</Steps>
